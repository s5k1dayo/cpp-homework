# 数据库这一块

## 学前问答

- 数据库(DB, Database)是什么?
    >用于在电脑中保存大量数据的一个地方。
    >
    >有**持久化储存** **有组织** **可共享** **减少冗余** 这几个基本性质。
- 数据库管理系统(DBMS, Database Manage System)是什么?
    >是一种对数据库进行各种操作和管理的程序
- 网络上经常说的 MySQL, Postgres, SQL Server, Oracle , 属于数据库还是数据库管理系统?
    >数据库管理系统
- Excel 是数据库吗?
    >不是，数据库可以进行多用户操作，建立并维护复杂关系，数据更多更复杂
- Excel 是数据库管理系统吗?
    >不是，*excel功能强大的电子表格软件，只能用于小规模数据分析，别用它来干DBMS的活儿*
